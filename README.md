# Library-Analytics-Pipeline-using-Databricks-and-PySpark

This project demonstrates a simple data engineering pipeline built using
Databricks and PySpark.

Pipeline:
books_raw → books_clean → books_transformed

Steps:
1. Data ingestion from CSV into Spark table
2. Data cleaning (null handling)
3. Data transformation (derived column - book_age)
4. Simple analytics using PySpark
